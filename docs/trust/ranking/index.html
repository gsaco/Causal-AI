Ranking Methodology

Trending is an attention signal derived from arXiv metadata, not a quality guarantee.

Signals

  • Recency score (exponential decay over 14 days)
  • Topic momentum (recent vs prior window)
  • Cross-listing signal (breadth across categories)
  • Version churn (revision activity, when available)

Formula

trending_score = 0.55 * recency +
                 0.20 * momentum +
                 0.15 * cross_list +
                 0.10 * churn

Weights are versioned and changes are logged in the update log.
